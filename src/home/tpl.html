如何发ajax

引用依赖：import {getJSON} from '../common/dataService';
方法名称：getJSON
参数：(url, params)
例子：
getJSON(IF_EXIST_NEW_MESSAGES,
     {userId : global.userId, reqTime: util.getTime()})
     .then(data => this.model.set({systemMessageCount: data, userId: global.userId}))
     .catch(e => alert(e));

如何增加router

在router.js中的routes增加route。 routers的key对应路由hash，value对应方法名称
例子：
routes: { "": "home", // #home "home": "home", // #home "manage/:id": "manage" // #manage/1 },
跳转到#manage，id=123

如何增加一个页面

在src中为每一个页面增加一个文件夹，对应hash的第一层。如#manage对应manage文件夹
在文件夹中，新建一个js和html，分别是脚本和模板内容
js和html的名字与文件夹保持一致，首字母大写

如何使用hogan

使用mustache-loader来处理模板，注意虽然名字是mustache-loader，但是使用的是hogan
例子：
var template = require('./Header.html');
var data = this.model.toJSON();
this.$el.html(template.render(data));
